# HTTP Server

A small Go HTTP server built as a learning project.

## Built so far

- Basic HTTP server
- Simple routing with `ServeMux`
- In-memory user storage
- User create, get, and delete handlers
- Token bucket rate limiting
- Middleware for rate limiting
- Basic request validation

## Ideal goals

- Custom router
- More middleware support
- Graceful shutdown
- Structured logging
- Better rate limiting control
- Benchmark report
